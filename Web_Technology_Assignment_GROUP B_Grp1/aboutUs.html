All Features About Audios and Videos

Audio:

  • Format: Audio files can be stored in various formats such as MP3, WAV, AAC, FLAC, etc. Each format has its own compression method and quality characteristics.
  • Bitrate: This refers to the amount of data processed per unit of time. Higher bitrate generally means better audio quality but larger file sizes
  • Sample Rate: It determines how many times per second the audio is sampled. Common sample rates include 44.1 kHz (CD quality) and 48 kHz (standard for digital audio).
  • Channels: Audio can be mono (single channel), stereo (two channels), or even surround sound (multiple channels).
  • Metadata: Information embedded within the audio file, such as artist name, album, track title, etc.
  • Compression: Some audio files are compressed to reduce file size. Lossy compression (like MP3) sacrifices some quality for smaller file sizes, while lossless compression (like FLAC) retains all audio data without loss.
  • Equalization (EQ): Adjustments made to frequency response to alter the balance of frequencies in audio.

Video:

  • Resolution: It indicates the number of pixels in each dimension that the video is composed of (e.g., 1920x1080 for Full HD).
  • Frame Rate: It denotes the number of individual frames or images displayed per second. Common frame rates include 24, 30, and 60 frames per second (fps).
  • Codec: Stands for coder-decoder. Codecs encode and decode digital data, and they can be lossy or lossless. Common video codecs include H.264, H.265 (HEVC), VP9, and AV1.
  • Aspect Ratio: It's the ratio of the width to the height of the video frame. Common aspect ratios include 16:9 (widescreen) and 4:3 (standard).
  • Bitrate: Similar to audio, video bitrate refers to the amount of data processed per unit of time. Higher bitrates generally mean better video quality but larger file sizes.
  • Container Format: The file format that holds the video data, audio data, subtitles, and metadata together. Examples include MP4, AVI, MKV, etc.
  • Compression: Video files are often compressed using codecs to reduce file size while maintaining visual quality.
